Best Free and Open Source Software

Here are the best free and open source tools I use, organized by category:

Development

  • Git - distributed version control system.
  • Visual Studio Code - extensible code editor.
  • Jupyter - interactive notebooks.

Robotics & Simulation

  • ROS - robotics middleware framework.
  • Webots - robot simulation software.
  • NVIDIA Isaac Sim - high-fidelity robotics simulator based on Omniverse for photorealistic and physics-accurate environments.
  • Gazebo - 3D robotics simulator.

CAD & Engineering

  • FreeCAD - parametric 3D CAD modeler.
  • LibreCAD - 2D CAD application.

Graphics & Design

  • GIMP - image editor.
  • Inkscape - vector graphics editor.
  • Blender - 3D creation suite.
  • Pinta - simple, open-source image editor inspired by Paint.NET.

Media

  • VLC - media player.
  • Shotcut - video editor.
  • LosslessCut - fast video trimming without re-encoding.
  • Audacity - audio recording and editing.
  • OBS Studio - video streaming and recording.

Office & Productivity

  • LibreOffice - office suite.
  • Zotero - reference management tool.
  • Okular - document viewer (PDF, EPUB, etc.).

Scientific & Engineering

  • GNU Octave - MATLAB-like numerical computing environment.
  • Scilab - numerical computation platform.
  • ParaView - scientific data visualization.

Networking & Browsers

  • Firefox - open-source web browser.
  • Tor Browser - privacy-focused browsing.
  • Remmina - remote desktop client.
  • qBittorrent - lightweight and open-source BitTorrent client.

Containers & Virtualization

  • Docker - containerization platform.
  • VirtualBox - virtualization software.

Terminal & Shell Tools

  • tmux - terminal multiplexer.

Security & Encryption

  • VeraCrypt - disk encryption software.
